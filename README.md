# Toll Management System

A desktop-based Toll Management System designed to streamline toll booth operations, reduce manual workload, and improve transaction efficiency. This solution automates vehicle toll collection, employee management, customer registration, pricing configuration, reporting, and record handling using a traditional yet reliable VB.NET and SQL Server architecture.
